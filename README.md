# Job Application Form

## Description
This project is a responsive **Job Application Form** developed using **HTML5** and **CSS3**. It provides a simple and user-friendly interface for applicants to submit their personal, educational, and professional details. The form is designed with a clean layout and is fully responsive, ensuring compatibility across desktop, tablet, and mobile devices.

## Features
- Responsive design
- User-friendly interface
- Personal information fields
- Educational details section
- Work experience section
- Skills and resume upload
- Submit and Reset buttons
- Form validation using HTML5

## Technologies Used
- HTML5
- CSS3

## Project Structure
```
project/
│── job-application.html
│── style.css
└── README.md
```

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open `job-application.html` in any modern web browser.

## Outcome
This project demonstrates the creation of a responsive and accessible job application form using HTML5 and CSS3, following modern web design practices.
